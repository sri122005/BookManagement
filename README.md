# Book Management System

A full-stack CRUD application built using Spring Boot, React, and MySQL.

## Tech Stack
Backend: Spring Boot, Spring Data JPA  
Frontend: React (Vite)  
Database: MySQL

## Features
- Add Book
- View Books
- Update Book
- Delete Book
- REST API Integration
- React UI

## Project Structure
BookManagement
├── demo (Spring Boot Backend)
└── book-ui (React Frontend)

## How to Run

Backend:
1. Run Spring Boot application
2. Server runs on http://localhost:8080

Frontend:
1. cd book-ui
2. npm install
3. npm run dev
4. Open http://localhost:5173
